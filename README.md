Titulo: Quiz Geografia

Introdução: Este projeto é um aplicativo de quiz interativo sobre Geografia, 
desenvolvido com o MIT App Inventor como parte da disciplina de 
Desenvolvimento de Aplicativos Móveis. O app apresenta 10 questões 
de múltipla escolha sobre temas como clima, relevo, hidrografia e 
geopolítica, com suporte a texto para fala (TTS) e feedback 
imediato ao usuário.
